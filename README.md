## 프로젝트 개요
- 프로젝트 명 : 모바일 웹 청첩장 프로젝트
- 참여 인원 : 1인
- 개발 기간 : 2022.11 (1개월)
- 특징 : 모바일 청첩장에 CRUDS 기능을 제공하는 게시판 추가, 하객들을 신랑측과 신부측으로 구분하여 축하메세지를 남길 수 있다. (+카카오맵 api를 이용하여 지도 구현)


## 사용 언어
- 
- FE : bootstrap, HTML/CSS, javascript
- BE : Java, Spring framework
- DB : My SQL, Heidi SQL, 
- API : Kakao Map
