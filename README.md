## 📌 프로젝트 개요
- **프로젝트 명:** 모바일 웹 청첩장 프로젝트  
- **참여 인원:** 1인  
- **개발 기간:** 2022.12.05 ~ 2022.12.17 (2주)  
- **주요 특징:**  
  - 모바일 환경에 최적화된 웹 청첩장 구현  
  - 게시판 기능(CRUDS) 제공  
  - 하객을 신랑/신부측으로 구분하여 메세지 작성 가능  
  - Kakao Map API로 지도 연동
